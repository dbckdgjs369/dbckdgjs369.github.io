## Lv 2. 최댓값과 최솟값

[문제](https://programmers.co.kr/learn/courses/30/lessons/12939)

풀이

    function solution(s) {
        let answer = '';
        let arr= s.split(' ').sort((a,b)=>a-b);
        answer+=arr[0]+" "+arr[arr.length-1];
        return answer;
    }
