## Lv 2. 큰 수 만들기

[문제](https://programmers.co.kr/learn/courses/30/lessons/42883)

    function solution(number, k) {
        let stack=[];
        let count=0;
        for(const num of number){
            while(count<k&&stack[stack.length-1]<num){
                stack.pop();
                count++;
            }
            stack.push(num);
        }

        while(count!==k){
            count++;
            stack.pop();
        }
        return stack.join("");
    }

풀이

큰 값이 나오면 이전 값 중 더 작은 값은 전부 버린다. => 스택 사용
